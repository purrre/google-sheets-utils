# Google Sheets Utilities Extension

Chrome extension that enhances Google Sheets with copy tools, popup suppression, and API helpers.

## Features
- Ctrl/Cmd + C Override - Copy text from protected/restricted sheets directly
- Popup Suppression - Autoclose Google Sheets popups
- API Helper - EZ generation of Sheets API v4 endpoints

## Installation
1. Open Chrome -> `chrome://extensions/`
2. Enable Developer mode
3. Click **Load unpacked** and select this folder
